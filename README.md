# YOUMELIST

## サイト概要
### サイトテーマ
「やりたいことリスト」を作って「ゆめ」を共有できるコミュニティサイト

### テーマを選んだ理由
今、SNS等の普及によって興味があることの情報収集が容易になり、  
今まででは考えなかったようなことも、「やってみたい」と思えることが増えたように感じます。  
そんな中で見つけた「自分のやりたいこと」を共有して、  
同じことに関心がある人や、実際に「やってみた」人からのコメントにより情報交換ができる事で、  
新しい目標を見つけたり、何か新しい事を始めるきっかけとなれば素敵だなと思います。

### ターゲットユーザ
* リストを作りたい、共有したい人
* 情報交換、進捗報告がしたい人

### 主な利用シーン
* やりたい事リストを作る時
* 情報交換、進捗報告に
