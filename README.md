# YOUMELIST

## サイト概要
### サイトテーマ
「やりたい事リスト」を作って共有できるコミュニティサイト

### テーマを選んだ理由
これまで、計画を立てたり、記録をつけることに何度も挑戦してきましたが、どれも続きませんでした。
唯一続いているのは、スマホのメモアプリに殴り書きしたメモをたまに見返すことだけなので、
メモのようにさっと書けて、気が向いたらみんなのゆめが見れるし、 自分のゆめもみんなに見てもらえるかもしれない。
そんなものがあれば楽しいかな、と思ったからです。

### ターゲットユーザ
リストを作りたい、共有したい人

### 主な利用シーン
やりたい事リストを作る時
